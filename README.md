# 多功能自定义相机-CaptureJz


## 特点功能:

 - 点击拍照，长按录制
 - 支持人脸识别
 - 支持美颜
 - 照片加滤镜
 - 视频加滤镜
 - 手动聚焦 和自动聚焦
 - 适配安卓全面屏
